# rust_fluent
====

![](https://travis-ci.org/takkanm/rust_fluent.svg?branch=master)

## Description

fluentd client for rust.

## Usage

Add this to your `Cargo.toml`:

```toml
[dependencies]
rust-fluent = "*"
```

and this to your crate root:

```rust
extern crate rust_fluent;
```

now only tcp with json  support.
